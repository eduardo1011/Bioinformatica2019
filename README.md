# Automatic_Notebook.ipynb
Esta bitácora electrónica está diseñada para hacer anotaciones funcionales por el método GO Slim haciendo uso de los recursos de UniProtKB y la Ontología Genética, es compatible con resultados provenientes de enfoques Genómicos, Transcriptómicos y Proteómicos. Como resultado de esta aplicación puedes obtener hasta 5 visualizaciones diferentes de los resultados funcionales.
